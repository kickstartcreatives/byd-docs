Stage 0 ~30 sec
Input Collection
Founder pastes any content that represents their brand. URL field is primary and prominent. Paste area is optional. No login. No required fields — one URL or any text is enough to start.
Website URL Bio Social captions Blog post Podcast transcript Video transcript ✦ new YouTube channel ✦ new Anything
UX principle
Generous, not interrogative. A single URL is enough to start.
Submit↓
Stage 1 <30 sec
Silent Analysis
One comprehensive AI pass. User sees streaming status messages — not a spinner. Outputs structured JSON. Never shown to user directly.
Reading your content… → Finding your voice… → Checking for gaps… → Ready.
Input gaps
Missing information
Info not provided — e.g. no origin story found. Flagged for inference or follow-up.
Brand gaps
Contradictions flagged
Things that don't logically fit — e.g. premium claim but low-tier pricing.
Updated v2.0
AI attempts to guess missing info (origin story, about page) wherever possible before routing to ask. Business idea validation added — gently flags fundamentally unviable ideas without full market analysis.
Analysis complete↓
Stage 2 ~2 min ✦ updated
What's Changing — 2 screens
Not a brand questionnaire. Gets to the heart of what's shifting. Each question answerable in a sentence.
Screen 1 — Required
Why a new site?
What's the thing your current site doesn't do or say?
+ Why now? (second required question on same screen)
Screen 2 — Optional
What should people feel or do?
When they land on the new site.
Skip button at bottom of screen.
Optional · skippable · does not block progression
Is there anything about your current positioning you want to move away from?
Updated v2.0
Screen 1 now asks both "Why a new site?" and "Why now?" as required questions. Screen 2 is fully optional with a visible Skip button — AI handles null answer gracefully via inference.
Answers submitted↓
Stage 2.5 Silent ✦ new
Reconciliation — silent
Runs silently during transition from Stage 2 to Stage 3. Reconciles founder's answers against Stage 1 JSON. Routes every gap. Produces structured input for Stage 3 readout. Latency target: under 10 seconds.
Alignments
Where answers confirm Stage 1
Tagged and passed to Stage 3 as confirmed signals.
Contradictions
Where answers conflict Stage 1
Recorded as facts — Stage 3 resolves, not Stage 2.5.
Gap routing — default to infer
Every Stage 1 gap gets routed: infer (default — draw conclusion from evidence), ask (only for true evidence voids), or defer (low-severity, doesn't change readout). "Where You're Headed" section verified to map coherently with Stage 2 answers.
Reconciliation complete↓
Stage 3 ~2–3 min
Strategic Readout
First thing the founder sees. Synthesized from Stage 1 + Stage 2 + Stage 2.5 reconciliation. Designed for skimming — bold lead-ins, key insight per section, full detail on expand.
USP Niche Offer Structure Voice & Tone Visual Direction Origin Story Brand Coherence Where You're Headed
Gap fill
AI infers missing content wherever possible. Only asks targeted follow-up if inference is genuinely not defensible. "Where You're Headed" explicitly maps to Stage 2 answers — no disconnect allowed.
Founder reviews↓
Stage 4 30 sec–2 min ✦ updated
Review & Confirm — three states
Founder approves, corrects, or fills gaps. Per-item checkboxes. Global approve button for lean path. Cascade only fires after all edits are complete — not after each individual edit.
State 1 — Correct
Approve with checkbox
Standard approval. No action needed beyond checking the box.
State 2 — Contradiction
Flagged + follow-up question
"What's your idea to fix this?" Skip option available.
State 3 — Missing info
Fill in, or let AI handle it
Text input + "Make your best guess" / "Use a placeholder" / "I'm not sure" buttons.
Lean path
"These all look great. Go. →"
Global approve button. Bypasses all individual checks. ~30 sec.
Depth path
Review section by section
Cascade fires once — after all edits complete, not after each one.
Updated v2.0 — cascade logic change
App waits for founder to complete all edits before running AI processing again. Prevents frustrating wait times between individual answers.
Confirmed↓
Stage 4.5 ~1–2 min ✦ new
The Aha! Moment
Intentionally slows the user down. Consolidates everything — original analysis plus corrections — in one emotional beat. Founder sees the full picture before the prompt generates. Designed to create confidence and validation.
Consolidated view
Everything in one place
USP, niche, voice, offer structure, corrections, origin story — all visible together.
Final tweak
Single edit box
"Any edits or last-minute notes / tweaks?" — communicates polish, not starting over.
Confirm button
"Yep, that's amazing. Build my prompt →" — emotional, confident, forward-moving.
Confirmed↓
Stage 5 Silent ✦ updated
Content Scaffolding + Prompt Generation
Runs internally. User sees engaging, celebratory loading text — not a spinner. All scaffolding generated, not collected.
Building your strategy… → Creating the money machine for you… → Almost there…
Pages scaffolded
Homepage → About → Services → Contact
Headline, hero copy, benefit statements, CTA logic per page.
Coherence check
About ↔ Home ↔ Services
Final alignment verified before output generates.
Scaffolding complete↓
Stage 6 ~30–60 sec ✦ updated
Output — The Money Screen
Highly congratulatory. Prominent copy-to-clipboard. Includes a few constructive notes about challenges ahead to keep in mind. White-label designer CTA at bottom.
Primary output
Claude Design–ready prompt
Positioning · niche · voice rules · visual direction · page-by-page copy · coherence rules.
Challenges ahead
A few things to watch for
Constructive notes — e.g. "Your offer ladder will need a front door — watch for this in the design."
Designer white-label CTA
"Need help turning this into a real site? Book a call." — founder has strategy doc in hand = warm lead.
