User action
Silent / internal
Decision point
Optional / skippable
Stage 0 ~30 sec
Input Collection
Founder pastes any content that represents their brand. URL field is primary and prominent. Paste area is optional. No login. No required fields.
Website URL Bio Social captions Blog post Podcast transcript Anything
UX principle
Generous, not interrogative. A single URL is enough to start.
Submit↓
Stage 1 <30 sec
Silent Analysis
One comprehensive AI pass. User sees streaming status messages, not a spinner. Outputs structured JSON — never shown to user directly.
Reading your website… → Finding your voice… → Checking for gaps… → Ready.
Preferred: single call
Positioning + voice
Niche, USP, offer structure, voice authority, market tension.
Fallback: parallel call B
Coherence + gaps
Cross-content alignment, contradictions, visual signals.
Gap-flagging logic
If evidence is missing for any field, mark it gap: true — don't fabricate, don't fail. Stage 3 asks targeted follow-ups for those gaps only.
Analysis complete↓
Stage 2 ~2 min
What's Changing
2 screens, 2 questions each. Not a brand questionnaire — these get to the heart of what's shifting. Each answerable in a sentence.
Screen 1
Why a new site, why now?
Strategic change or just aesthetic?
Screen 2
What should people feel or do?
When they land on the new site.
Optional · skippable · does not block progression
Is there anything about your current positioning you want to move away from?
Answers submitted↓
Stage 3 ~2–3 min
Strategic Readout
First thing the founder sees. Synthesized from Stage 1 + Stage 2. Designed for skimming — bold lead-ins, key insight per section, full detail on expand.
USP Niche Offer Structure Voice & Tone Visual Direction Origin Story Brand Coherence Where You're Headed
Gap fill
If a section can't be populated, tool asks one targeted question — not a new form. Most common: About page origin story. Framed as "we need this one thing."
Founder reviews↓
Stage 4 30 sec–2 min
Validate
Founder confirms, edits, or redirects. Nothing moves to Stage 5 until explicitly confirmed — but confirming doesn't require editing.
Lean path
"Looks right, continue →"
One click. No cascade fires. ~30 sec. This is the fundamental promise.
Depth path
Edit sections individually
Cascade logic fires only on edit.
Cascade logic — fires on edit only
USP change → updates voice, offer ladder, homepage promise.
Niche change → updates ideal client, offer positioning, visual direction.
Origin story change → updates About↔Home alignment.
Fallback: flag stale sections rather than auto-propagate.
Confirmed↓
Stage 5 Silent
Content Scaffolding
Runs internally between Stage 4 and Stage 6. User is not involved. All scaffolding is generated, not collected.
Pages scaffolded
Homepage → About → Services → Contact
Headline, hero copy, benefit statements, CTA logic per page.
Coherence check
About ↔ Home ↔ Services
Alignment verified before output generates.
Scaffolding complete↓
Stage 6 ~30–60 sec
Output — The Money Screen
One complete prompt ready to paste into Claude Design. Prominent copy-to-clipboard. Review is optional.
Claude Design–ready prompt
Positioning · niche · voice rules · visual direction · page-by-page copy direction · coherence rules
Copy to clipboard
Designer white-label CTA
"Need help turning this into a real site? Book a call." — founder has strategy doc in hand = warm lead.
Lean path time budget — under 10 min is non-negotiable
